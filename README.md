# Молочные Фермы Ильченко

Семейная молочная ферма с полным циклом производства натуральных молочных продуктов. Современный одностраничный сайт с интерактивной картой магазинов, многоуровневой системой отправки форм и расширенной документацией.

## Описание

Проект представляет собой современный веб-сайт для семейной молочной фермы, основанной в 1998 году. Сайт сочетает традиционные методы производства молочных продуктов с передовыми веб-технологиями и обеспечивает полный пользовательский опыт от презентации продукции до оформления заказов.

Ключевые особенности:
- Полностью адаптивный дизайн для всех устройств
- Интерактивная карта с 7 фирменными магазинами
- Двойная система отправки форм (Telegram + Email)
- Система сохранения UTM-параметров в localStorage
- Современная анимация и микроинтеракции
- Полная правовая документация (политика конфиденциальности, пользовательское соглашение)
- Оптимизация производительности и доступности

## Технические особенности

### Современные веб-технологии

Проект использует передовые веб-стандарты и современные API:

- **[Intersection Observer API](https://developer.mozilla.org/en-US/docs/Web/API/Intersection_Observer_API)** для плавной анимации элементов при прокрутке
- **CSS Custom Properties** для гибкой системы дизайна
- **CSS Grid и Flexbox** для адаптивной верстки
- **ES6+ JavaScript** с современными паттернами проектирования
- **Debounce и Throttle** функции для оптимизации производительности
- **Lazy Loading** для изображений
- **localStorage API** для сохранения пользовательских данных

### Необычные технические решения

- **Двойная система отправки форм** (Telegram + Email) для надежности доставки
- **Система сохранения UTM-параметров** в localStorage для аналитики конверсий
- **Яндекс.Карты API** с кластеризацией и геолокацией
- **Кастомная система уведомлений** без зависимостей от библиотек
- **Progressive Enhancement** - сайт работает без JavaScript
- **Skeleton Loading** для улучшения UX при загрузке данных
- **Многоуровневая валидация форм** с визуальной обратной связью

### Внешние зависимости

- **[Inter](https://fonts.google.com/specimen/Inter)** и **[Playfair Display](https://fonts.google.com/specimen/Playfair+Display)** - шрифты от Google Fonts
- **[Material Symbols Outlined](https://fonts.google.com/icons)** - иконки от Google
- **[Яндекс.Карты API](https://yandex.ru/dev/maps/jsapi/)** - картографический сервис
- **[IMask](https://imask.js.org/)** - маскировка полей ввода

## Структура проекта

```
.
├── index.html                  # Главная страница сайта
├── privacy-policy.html         # Политика обработки данных
├── terms-of-service.html       # Пользовательское соглашение
├── Адресса.md                  # Список адресов магазинов (Markdown)
├── css/
│   ├── styles.css              # Основные стили и дизайн-система
│   ├── about-section.css       # Стили раздела "О нас"
│   ├── products-section.css    # Стили раздела продукции
│   ├── stores-section.css      # Стили раздела магазинов
│   ├── feedback-section.css    # Стили формы обратной связи
│   ├── footer.css              # Стили подвала
│   └── privacy-policy.css      # Стили правовых документов
├── js/
│   ├── script.js               # Основная логика и компоненты
│   ├── map.js                  # Работа с Яндекс.Картами
│   └── form.js                 # Обработка форм с localStorage
├── php/
│   ├── send.php                # Отправка данных в Telegram
│   └── send-email.php          # Отправка данных на Email
├── data/
│   └── stores.json             # Данные о магазинах (JSON)
└── img/
    ├── favicon/                # Фавиконы разных размеров
    ├── stores/                 # Фото магазинов
    ├── logo.svg                # SVG логотип компании
    └── [другие изображения]
```

### Ключевые директории

- **`css/`** - модульная система стилей с разделением по компонентам и страницам
- **`js/`** - объектно-ориентированный JavaScript с паттернами проектирования и localStorage интеграцией
- **`php/`** - двухуровневая система обработки форм (Telegram + Email)
- **`data/`** - структурированные JSON-данные для динамической загрузки контента
